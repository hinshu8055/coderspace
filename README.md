# 🚀 Coderspace – Real-Time Collaborative Coding Platform

Coderspace is a **real-time collaborative coding platform** that enables multiple developers to write, edit, and execute code together seamlessly.
It provides a shared coding environment where developers can collaborate efficiently with live updates and instant code execution.

## ✨ Features

* 🔄 **Real-Time Collaboration** – Multiple users can edit code simultaneously
* ⚡ **Live Code Execution** – Run code instantly with support for multiple languages
* 👥 **User Presence Indicators** – See who is currently active in the room
* ⌨️ **Typing Indicators** – Know when other developers are typing
* 🧠 **Multi-language Support** – Execute code in different programming languages
* 🌐 **Responsive UI** – Smooth and interactive coding experience

## ⚙️ Installation & Setup

Clone the repository:

```
git clone https://github.com/hinshu8055/coderspace.git
```

Go to the project directory:

```
cd coderspace
```

Install dependencies:

```
npm install
```

Start the backend server:

```
node server.js
```

Open the frontend in your browser and start collaborating.
